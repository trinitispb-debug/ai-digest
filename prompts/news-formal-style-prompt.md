# Промпт: новостная статья в формальном стиле

## Контекст

Дайджест новостей по промышленной метрологии.
Аудитория: инженеры-метрологи, технологи, специалисты КИМ и 3D-сканирования.
Формальный стиль: деловой тон, третье лицо, без разговорных оборотов.

## Задача

Напиши обзор новости из области промышленной метрологии.
Краткий пересказ: суть события, технические характеристики и отличия от аналогов, практическая значимость для метрологического обеспечения производства.

## Формат

- 300 слов (не больше).
- Заголовок: до 80 символов, конкретный, без кликбейта.
- 2-3 подзаголовка.
- Markdown, без HTML.
- Ссылка на источник в конце.

## Ограничения

- Третье лицо («компания представила», «прибор обеспечивает»).
- Деловой тон, без разговорных и оценочных оборотов.
- Терминология по ГОСТ/МДМВ (не кальки: «координатно-измерительная машина», а не «CMM»).
- Без маркетинга, без эмодзи.
- Технически точное изложение.
