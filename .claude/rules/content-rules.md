# Правила контента

- Язык: русский
- Терминология ГОСТ/МДМВ, не кальки (КИМ, а не CMM; поверка, а не calibration)
- 300-500 слов на статью
- Markdown, без HTML
- Без эмодзи
- Без маркетинговых оборотов
- Ссылка на источник обязательна
