# CLAUDE.md

## Проект

**Metrology Digest** — дайджест новостей по промышленной метрологии.

- Тематика: линейно-угловые измерения, контроль геометрии деталей, лазерное 3D-сканирование, реверс-инжиниринг, КИМ (координатно-измерительные машины), поверка и калибровка
- Стек: Astro 6, TypeScript, MDX, Vercel
- Контент: русский язык
- Промпты для генерации статей лежат в `prompts/`

## Структура

- `src/content/blog/` — статьи дайджеста (markdown/mdx)
- `src/pages/` — маршруты (главная, блог, RSS)
- `prompts/` — промпты для генерации контента
- `public/` — статические файлы

## Правила

- Все статьи на русском языке
- Терминология ГОСТ/МДМВ, не кальки с английского
- Без эмодзи в статьях
- Markdown, без HTML
